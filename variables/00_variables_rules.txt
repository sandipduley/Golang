1. A variable name must start with a letter or an underscrore character ( _ )
2. A variable name cannot start with a digit
3. A variable name can only contain alpha-numeric characters and unserscore (a-z, A-Z, 0-9, _)
4. Variable names are case sensitive
5. There is no limit on the length of the variable name 
6. A variable name cannot contain spaces
7. A variable name cannot be any GO keywords
